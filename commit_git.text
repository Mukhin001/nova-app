Примеры типов:

feat — новая функциональность (feature)

fix — исправление ошибки (bugfix)

docs — изменения в документации

style — форматирование, исправление пробелов, отступов

refactor — рефакторинг без изменения логики

test — добавление или изменение тестов

chore — вспомогательные изменения (например, настройки сборки)
